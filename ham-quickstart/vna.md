# 介绍
矢量网络分析仪（Vector Network Analyzer，简称VNA）是一种电磁波能量测试设备，主要用于测量射频微波器件、天线、电缆等电子元器件在宽频带内的散射参数（S参数），涵盖频率范围从千赫兹至太赫兹级。其核心应用领域包括无线通信、雷达系统、航空航天及材料科学研究，可对无源/有源器件的传输反射特性进行精确量化评估。
